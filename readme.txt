== Description ==

WooVirtualWallet is a free plugin for WooCommerce that allows your customers to stock money on a virtual wallet. They can use the wallet's money to pay their orders.

= WooVirtualWallet Features =

* Customers can pay their orders with their wallet's money
* Give Cashback on the Wallet for purchasing products
* Edit users wallet amount
* Prevent the use of wallet to buy products giving wallet money
* Wallet Payment Gateway
* Translated in multiple languages : English, French
* WPML Compatibility

For more information, visit our website [Long Watch Plugins - Documentation](https://plugins.longwatchstudio.com/en/documentation-en/woovirtualwallet/)

== WooVirtualWallet Pro ==

**[WooVirtualWallet Pro](https://plugins.longwatchstudio.com/en/product/woovirtualwallet-en/)** adds a ton of new features to WooVirtualWallet:

* Let customers partially pay orders with their wallet's money
* Gems Mode : Use your own currency on your website
* Refund orders directly on the wallet
* Widgets/Shortcodes to display the various wallet information
* Customer feedback on WooCommerce's cart and checkout pages
* Restrict the usage of wallet to specific products
* Limit partial payment to a percentage of cart total
* Menu entry point with wallet money
* Wallet movement history
* WooCommerce Subscriptions support
* New features for WooRewards
* Translated in multiple languages : English, French
* WPML Compatibility

= Gems Mode - An exclusive feature =

Create a side shop within your main store and use your own currency for the shop's products.
* Let customers buy or earn gems, diamonds, points or whatever currency you can think of.
* New types of products in WooCommerce to create products that can only be purchased with your currency.
* Possibility to use the currency to get immediate discounts on regular orders.
* Give cashback in your currency when customers buy regular products.
* **Create an exclusive Gift Shop where customers can spend their gems to get exclusive products**


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/woovirtualwallet` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->WooVirtualWallet screen to configure the plugin
